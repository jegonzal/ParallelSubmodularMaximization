  4. abstract class ConcurrencyControl_SubmodularDoubleGreedy extends AnyRef

    Concurrency Control Double Greedy (CC-2g) algorithm.

    Concurrency Control Double Greedy (CC-2g) algorithm. CC-2g runs the double greedy algorithm in parallel, coordinating between threads to ensure serializability. The approximation factor of CC-2g is the same as double greedy due to serial equivalence of the outcome. CC-2g does not scale as well as CF-2g because of the additional coordination required.
